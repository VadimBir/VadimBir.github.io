KV Cache Eviction Policy for Large Language Models

Technologies: Python, vLLM Framework, POS Tagging, Memory Optimization

Engineered a novel approach to KV cache eviction policy using POS tagging to determine token relevance, effectively optimizing memory footprint and performance improvement of LLMs. The strategy ensures performance improvement and resource management within a vLLM framework via a custom BlockCombiner class prioritizing essential tokens for model accuracy.

Password Manager (Django)

Technologies: Django, Python, Web Framework, Authentication, Database Security

A working password manager using Django web framework. This website allows users to register, login and logout. The web system enables users to add password name tags, passwords, and descriptions. Passwords are visible only to the author, ensuring secure personal data management.

Designed and developed a 3D Connect 4 game in Unity with a three-axis board. Created custom scripts in C# that handle all calculations including win/lose condition checks, free-cam script, border limits, and teleportation functionality.

Battleship Game with Design Patterns

Technologies: Java, Design Patterns, JUnit, Object-Oriented Programming

Battleship game in Java implementing four design patterns: Decorator Pattern, Factory Pattern, Command Pattern, and State Pattern. Features include game saving mechanics, resume functionality after game closure, and extensible addon architecture. Includes comprehensive JUnit tests for reliability.

Password Manager (Java)

Password Manager written in Java using object-oriented design principles. Allows users to create passwords with tags, perform partial searches for saved passwords via name tags, and delete passwords using name tag identification.
