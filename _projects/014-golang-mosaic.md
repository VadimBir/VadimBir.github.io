A <strong>Go-hosted</strong> mosaic webpage utilizing <strong>Go's concurrency</strong> model for request handling alongside <strong>HTML and CSS frontend</strong>. Generates a 6×6 grid of random shapes, colors, and characters with dynamic re-randomization and responsive sizing. Demonstrates Go's <code>net/http</code> <strong>concurrent request serving</strong> and <strong>template rendering pipeline</strong>.
